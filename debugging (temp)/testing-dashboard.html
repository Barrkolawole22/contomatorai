🚀 Local Setup Guide

Backend Setup:

cd backend
npm install
npm run dev

Frontend Setup:

cd frontend
npm install
npm run dev

Environment Variables:

# Backend .env
MONGODB_URI=mongodb://localhost:27017/ai-content-platform
JWT_SECRET=your-jwt-secret
PORT=5000

# Frontend .env.local
NEXT_PUBLIC_API_URL=http://localhost:5000/api
